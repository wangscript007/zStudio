文件名格式：“<数据模型名>.json”
文件编码格式应为如下图：
以下替换“<>”内容为实际应用的内容，文件内容格式如下： 
                    
[
   {
      "id":"<字段名>",
      "name":"<字段显示名>",
      "default":"<默认值>",
      "controlType":"<控件类型>",
      "uiVisible":"<可显示>",
      "layout":"<布局类型>",
      "dataBlock":"<所属面板域，面板域值为“整数”，面板域的顺序为“整数升序排列”>"
   }
]
注意：
“controlType”的值应为“1”、“2”、“3”、“4”、“5”、“6”、“7”、“8”。

“layout”的值应为“0”、“1”和“2”。
